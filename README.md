## GraphiQL  Script
- Get all Events only
	query{
	events {
		title
		price
	}
	}
- Get Events with user's detail
	query{
		events{
					creator{
						email
						createdEvents{
							title
							creator{
							email
							}
						}
				}
			}
	}
- Get user and event details through booking id
query{
  bookings{
    _id
    createdAt
    event{
      title
      creator{
        email
      }
    }
  }
}
